This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Installation

Ce projet a été développé avec **pnpm**. Il est recommandé d'utiliser `pnpm install` pour garantir la cohérence des dépendances.

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Running the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
